# Páginas da interface

## Login

Campos:

- Email.
- Senha.

Ações:

- Entrar.

## Dashboard

Indicadores:

- Vendas do mês.
- Variação de vendas.
- Gastos com ingredientes.
- Gastos com embalagens/adesivos.
- Lucro bruto estimado.
- Ticket médio.
- Produtos mais vendidos.
- Produtos com maior e menor margem.

## Ingredientes

Funcionalidades:

- Listar.
- Buscar.
- Filtrar por categoria.
- Criar.
- Editar.
- Inativar.
- Ver custo médio.
- Ver estoque atual.

## Compras

Funcionalidades:

- Listar compras.
- Registrar nova compra.
- Adicionar múltiplos itens.
- Separar ingredientes e materiais.
- Ver total da compra.

## Embalagens e adesivos

Funcionalidades:

- Listar.
- Criar.
- Editar.
- Inativar.
- Ver estoque.
- Ver custo unitário.

## Produtos

Funcionalidades:

- Listar produtos.
- Criar produto.
- Editar produto.
- Informar ingredientes.
- Informar materiais.
- Informar rendimento.
- Informar perda.
- Informar margem.
- Ver custo calculado.
- Ver preço sugerido.
- Definir preço manual.

## Cardápio diário

Funcionalidades:

- Escolher data.
- Criar cardápio.
- Adicionar produtos.
- Habilitar produto.
- Desabilitar produto.
- Alterar preço do dia.
- Publicar cardápio.
- Encerrar cardápio.

## Vendas

Funcionalidades:

- Registrar venda.
- Selecionar produtos.
- Informar quantidade.
- Aplicar desconto.
- Escolher pagamento.
- Escolher canal.
- Ver total.
- Listar vendas do dia.

## Logs

Funcionalidades:

- Listar logs.
- Filtrar por entidade.
- Filtrar por usuário.
- Filtrar por ação.
- Filtrar por período.
