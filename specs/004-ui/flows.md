# Fluxos da interface

## Fluxo 1 — Cadastro inicial

1. Usuário faz login.
2. Usuário cadastra ingredientes.
3. Usuário cadastra embalagens e adesivos.
4. Usuário registra compras iniciais.
5. Sistema calcula custos.
6. Usuário cadastra produtos.
7. Sistema sugere preços.

## Fluxo 2 — Registro de compra

1. Usuário acessa Compras.
2. Clica em "Nova compra".
3. Informa data e fornecedor.
4. Adiciona itens.
5. Informa quantidade, unidade e valor.
6. Confirma.
7. Sistema atualiza estoque e custos.
8. Sistema mostra confirmação.

## Fluxo 3 — Cadastro de produto

1. Usuário acessa Produtos.
2. Clica em "Novo produto".
3. Informa nome e categoria.
4. Adiciona ingredientes.
5. Adiciona embalagens/adesivos.
6. Informa rendimento.
7. Informa perda.
8. Informa margem.
9. Sistema exibe custo e preço sugerido.
10. Usuário define preço final.
11. Salva produto.

## Fluxo 4 — Cardápio diário

1. Usuário acessa Cardápio.
2. Escolhe a data.
3. Sistema mostra cardápio existente ou botão para criar.
4. Usuário adiciona produtos.
5. Usuário habilita/desabilita itens.
6. Usuário ajusta preço do dia.
7. Usuário publica.

## Fluxo 5 — Registrar venda

1. Usuário acessa Vendas.
2. Clica em "Nova venda".
3. Seleciona produtos.
4. Informa quantidade.
5. Informa desconto, se houver.
6. Escolhe canal e pagamento.
7. Sistema calcula total.
8. Usuário confirma.
9. Sistema salva venda e snapshots.

## Fluxo 6 — Analisar dashboard

1. Usuário acessa Dashboard.
2. Sistema mostra mês atual.
3. Usuário pode trocar mês/ano.
4. Sistema compara com mês anterior.
5. Sistema mostra variações.
