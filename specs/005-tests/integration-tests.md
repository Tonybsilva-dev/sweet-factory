# Testes de integração

## Ingredientes + Compras

Cenário:

1. Criar ingrediente.
2. Registrar compra.
3. Verificar estoque.
4. Verificar custo médio.
5. Verificar log.

## Embalagens + Compras

Cenário:

1. Criar embalagem.
2. Registrar compra.
3. Verificar estoque.
4. Verificar custo unitário.
5. Verificar log.

## Produtos

Cenário:

1. Criar ingredientes.
2. Registrar compras.
3. Criar embalagem.
4. Registrar compra de embalagem.
5. Criar produto.
6. Verificar custo calculado.
7. Verificar preço sugerido.

## Cardápio

Cenário:

1. Criar produto.
2. Criar cardápio.
3. Publicar cardápio.
4. Desabilitar produto.
5. Verificar logs.

## Vendas

Cenário:

1. Criar produto.
2. Criar cardápio.
3. Registrar venda.
4. Verificar snapshots.
5. Verificar lucro estimado.
6. Verificar log.

## Dashboard

Cenário:

1. Registrar compras no mês anterior.
2. Registrar compras no mês atual.
3. Registrar vendas no mês anterior.
4. Registrar vendas no mês atual.
5. Consultar dashboard.
6. Verificar variações.
